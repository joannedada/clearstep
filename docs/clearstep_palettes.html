ClearStep — Accessibility Colour Profiles

Microsoft AI Innovation Challenge 2026 — Neurodivergent Accessibility Documentation

All profiles meet WCAG 2.1 AA contrast requirements (4.5:1 for normal text)

These aren't themes — they're cognitive accessibility profiles.

Each one is designed for a different type of visual processing need. ADHD, autism, and dyslexia can overlap but their visual needs differ significantly person to person. ClearStep's approach: accessible defaults + saved user preferences + low-stimulation presets. The system remembers and applies the selected profile automatically across sessions — no reconfiguration needed.

There is no single best colour set for all neurodivergent users. That's the point. One size does not fit all — and this system was designed from the start to reflect that.
